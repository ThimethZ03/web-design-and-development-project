View Validation

SVG Implementation

Explanation of how the sitemap is visually displayed using SVG.

Key HTML and SVG Elements

  • <svg> – SVG container
  • <g> – Groups elements
  • <circle> and <rect> – Page nodes
  • <line> and <path> – Connections between nodes
  • <text> – Node labels
  • <a> – Makes nodes clickable
  • <title> – Provides accessibility info

CSS Styling for SVG

  • Responsive Design – Using viewBox
  • Hover Effects – Change fill color on hover
  • Accessibility – Focus states for keyboard users
  • Color Coding – Visual difference between node types
.sitemap-node { fill: #6B9B7A; stroke: #4A7C59; stroke-width: 2; transition: all 0.3s ease; cursor: pointer; } .sitemap-node:hover { fill: #FF8C42; transform: scale(1.1); } .sitemap-node:focus { outline: 2px solid #FF8C42; outline-offset: 2px; }

Accessibility Features

  • Alt Text – Via title elements
  • Keyboard Navigation – Focusable nodes
  • Screen Reader Support – ARIA roles/labels
  • Color Contrast – Meets WCAG 2.1 standards
